What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web browsers to interact with the structure and content of HTML documents. It provides a way to access and manipulate the document's elements, attributes, and text.

How do you select an element from the DOM?

Selecting an element from the DOM involves using the document object's methods and properties. You can use the `getElementById()` method to select an element by its id, the `getElementsByClassName()` method to select elements by their class name, and the `getElementsByTagName()` method to select elements by their tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you add a single event listener to a parent element instead of individual child elements. When the event occurs on the parent element, the event handler is triggered on all descendant elements that match the event's target. This is useful when you want to handle events for multiple elements that share a common functionality.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element's attributes and styles using the DOM involves using the element's properties and methods. You can set the value of an element's attribute using the `setAttribute()` method, get its attribute value using the `getAttribute()` method, and remove an attribute using the `removeAttribute()` method. You can modify an element's style using the `style` property, set individual CSS properties using the dot notation (e.g., `element.style.color = "red"`), or use CSS classes and inline styles.
